Node

É um ambiente de execução de javascript que permite que o codigo js seja executado em nosso pc.

Existem outros ambientes:
Deno

Deno é um ambiente novo mas com uma nova perspectiva de segurança. O criador é um dos pais do NodeJS.

NPM - 

Node Package Manager (Gerenciador de pacotes do Node) - Gerencia os pacotes de módulos 
Ajuda a instalar e desinstalar pacotes e gerenciar versões e dependencias necessárias para executar um projeto.
